
# 🧩 Maze Solver
Maze solver using A* and Dijkstra algorithms with priority queue
## 📌 Description
This project implements a maze-solving algorithm using **A\*** and **Dijkstra’s Algorithm** to find the shortest path in a grid-based maze.

## ⚙️ Features
- Shortest path finding using **Dijkstra’s Algorithm**  
- Heuristic-based optimization using **A\***  
- Efficient node selection using a **priority queue (min-heap)**  
- Graph-based representation of the maze  
- Comparison between different pathfinding approaches  

## 🧠 Algorithms Used
- Dijkstra’s Algorithm  
- A\* Algorithm  
- Priority Queue (Min Heap)  
- Heuristic Function (for A\*)  

## 🛠️ Technologies
- C++  
- Data Structures & Algorithms
